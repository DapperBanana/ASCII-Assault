# ASCII Assault - Project Plan

## Goals

*   **Core Gameplay Loop:** Implement a basic, playable ASCII-based arena combat game.
*   **Networking:** Establish client-server communication for real-time multiplayer.
*   **Authentication:** Secure user accounts with password hashing.
*   **Database Integration:** Store user data (accounts, stats) in a MySQL database.

## Next Steps

1.  Implement database user authentication.
2.  Implement chat functionality.
3.  Implement the core game loop.
